# CodeCrest

A web-based interactive code editor platform which enables users to write, view, and save real-time code snippets. Offers real-time rendering of your HTML, CSS, and JS code,
promoting instant feedback facilitating debugging and design adjustments. Users can login to save and share their work with the community and can also interact with othersâ€™ work, fostering a
collaborative learning environment.

## Tech stack:

- JavaScript
- React
- Tailwind CSS
- Redux
- Firebase

## Steps to run project:

```
yarn start
```
